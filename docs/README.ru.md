### Чем занимаюсь

- Telegram-first продукты с веб-кабинетами, инструментами для команды и реальной бизнес-логикой.

### Избранные проекты

#### Приватные продукты

- **BadVPN** — production VPN-платформа с Telegram- и веб-флоу, multi-auth, платежами, рефералкой, AI-поддержкой и насыщенной доменной логикой.
- **MegaFon Support Bot** — автоматизация клиентской поддержки через Telegram-бот и веб-админку, с пайплайнами адресов и тарифов, геокодингом и fuzzy search.
- **Badass Bot** — Telegram-магазин с deep-link навигацией, кешированием изображений, экспортом заказов, рассылками и admin-операциями.
- **Клиентские веб-системы** — headless commerce и B2B-платформы на Astro и Payload CMS, включая миграции с legacy-стеков.

#### Публичные проекты

- [`CareerTrack`](https://github.com/Rerowros/mtc) — Telegram-first AI-карьерный ассистент со структурированным онбордингом, ранжированием вакансий и companion web app.
- [`Remoji-TG-MCP`](https://github.com/Rerowros/Remoji-tg-mcp) — MCP-сервер для поиска Telegram emoji и стикеров через browser-first auth и визуальный выбор.
- [`PasarGuard Node Bridge`](https://github.com/Rerowros/node_bridge_py) — асинхронный Python-бридж для управления node-инфраструктурой по gRPC и REST.
- [`kworker`](https://github.com/Rerowros/kworker) — async Python-клиент для API Kwork, оформленный как отдельная библиотека.
